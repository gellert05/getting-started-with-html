1.1 The web

The web contains HTML documents that glue everything together. And the links - hyperlinks - make the web "weblike". The was web invented in the early 1990s and now has around 30 billion pages.

The major pieces of the web:

  • You: your computer & browser
  • The internet
  • Servers

The browser send requests to server using a protocol called HTTP (hyper text transfer protocol )and the server responds with files that the browser displays.

1.2 HTML

HTML (hyper text markup language) document are the heart of the web.

HTML is made up of:

  • Text content what you see
  • Markup what it looks like
  • References to other documents e.g. images, videos
  • Links to other pages

1.2.1. HTML Markup

HTML Markup made up of TAGS.

This is normal text

  • <b>and this tag will make your text bold</b>
  • <em>this is emphasized text in italics</em>
  • <p>this tag is used for creating paragraph</p>

This tag will put an image into the HTML document, with a set width and height and 2 px border:

  • <img src="Embedded URL" width="500" height="250" border="2" alt="alternative text& in lieu of the image" title="this is optional write here the image title if you wish"> - void tag*
  • <br> - another void* tag. This one breaks the lines.

*Void tag: the tag has no closing tag, so it is called a VOID TAG.

The whole list of tags can be found here: http://www.w3schools.com/tags/default.asp
